# Contributing

Thank you for your interest in this repository.

This repository serves primarily as a professional technical portfolio documenting laboratory science, industrial microbiology, quality systems, and applied analytical projects.

## Reporting Issues

If you identify an error or have a suggestion for improvement, please open a GitHub Issue with:

- A clear description of the issue.
- Steps to reproduce (if applicable).
- Supporting references where appropriate.

## Pull Requests

External pull requests are not currently accepted.

Suggestions, corrections, and technical discussions are welcome through GitHub Issues.

## Professional Standards

All contributions and discussions should:

- Be evidence-based.
- Maintain scientific accuracy.
- Respect intellectual property.
- Use professional language.
- Cite appropriate references when applicable.

Thank you for helping improve the quality of this portfolio.
